COMMAND FLOW:

    User Types Command in Control GUI
              ↓
    Command sent to Spacecraft Simulator
              ↓
    Logged in EVR
              ↓
    (Future: Command execution affects spacecraft state)


EMERGENCY RESPONSE:

    User Clicks Red TLM Display in EID
              ↓
    Anomaly Response Window Opens
              ↓
    Options: Safe Mode, Power Reset, Attitude Hold, etc.
              ↓
    User selects response
              ↓
    Command sent to spacecraft
              ↓
    Event logged in EVR


TYPICAL ORBIT CYCLE (92 minutes):

    T+0:00  - Start over Pacific Ocean
    T+23:00 - Day side, battery charging, solar high
    T+46:00 - Enter eclipse, solar drops to zero
    T+46:01 - Battery starts discharging (CAUTION)
    T+46:30 - Battery voltage drops below 26V (YELLOW)
    T+69:00 - Exit eclipse, solar returns
    T+69:01 - Battery starts charging
    T+92:00 - Complete one orbit, back to start


FILE USAGE GUIDE:

    For First Time:
    - Read: README.md (comprehensive guide)

    - Check: requirements.txt (dependencies)


SYSTEM STATES:

    NOMINAL:
    - All displays GREEN
    - Battery 26-28.5V
    - Attitude within ±2°
    - Temperature 18-28°C
    - Solar array active (day side)
    
    CAUTION:
    - Some displays YELLOW
    - Battery 24.5-26V
    - Attitude 2-4°
    - Needs monitoring
    
    WARNING:
    - Some displays RED
    - Battery <24.5V
    - Attitude >4°
    - Immediate action required
    - Consider anomaly response
